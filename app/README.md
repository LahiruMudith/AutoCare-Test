# HelloJavaApp

A simple Android mobile app written in **Java** using **Android Studio**, **Android Gradle Plugin 8.2.2**, and **Gradle 8.5**.  
This is a starter template with a single activity (`MainActivity`) and a `ConstraintLayout` UI.
